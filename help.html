  1. How to start building the Risk Table ?
    • Click the lets starticon at the home page.
    • Enter all the fields displayed for building the table.
    • Upload a csv file if you wish to include your own table by clicking on the choose file tab.

  2. What is a Risk Table Builder in Software Engineering ?

    3. A Risk in Software Engineering is an expectation of loss, generally caused due to lack of the required resources for a certain software or the time required to implement features within the software.


  3. What is Risk Impact ?

    4. A Risk Impact is an estimate of the potential losses that are associated within a Risk that has been identified.


  4. What is a Risk Probability ?

    5. A Risk Probability is an estimate for the percentage of likeness that the risk might occur.


  5. What is a Risk Category ?

    6. A Risk Category is the description of the risk in terms of which level it should be specified, it can be Insignificant, Minor, Moderate, Major, Catastrophic (each of these categories are listed in increasing order of impact).


  6. Software does not show the information stored in file ?

    7. Make sure the file is uploaded in csv format.


  7. How to return to home page ?

    8. Click the icon on the top left corner of the page.


  8. Information for database file not displaying?

    9. Entries are case sensitive.
